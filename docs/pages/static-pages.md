## 2. Help Center / FAQ

### URL: `/help`

```
┌────────────────────────────────────────────────────────────────┐
│                                                                │
│  Help Center                                                   │
│                                                                │
│  [──── Search help articles... ────]                           │
│                                                                │
│  ── Popular Topics ───────────────────────────────────────     │
│                                                                │
│  ┌──────────────┐  ┌──────────────┐  ┌──────────────┐         │
│  │ 🛒 Ordering  │  │ 🚚 Delivery  │  │ 💵 Payments  │         │
│  └──────────────┘  └──────────────┘  └──────────────┘         │
│  ┌──────────────┐  ┌──────────────┐  ┌──────────────┐         │
│  │ 👤 Account   │  │ 🏪 Selling   │  │ 🔄 Returns   │         │
│  └──────────────┘  └──────────────┘  └──────────────┘         │
│                                                                │
│  ── Frequently Asked Questions ───────────────────────────     │
│                                                                │
│  ▼ How does cash on delivery work?                             │
│    When your order arrives, you pay the delivery driver        │
│    the total amount in cash. No online payment needed.         │
│                                                                │
│  ▼ How long does delivery take?                                │
│    Same city: 1-2 days. Same province: 2-4 days.              │
│    Different province: 4-7 days.                               │
│                                                                │
│  ▼ How do I return a product?                                  │
│    Contact the seller through the messaging system within      │
│    3 days of delivery. If approved, the driver will pick       │
│    up the item and you'll receive a cash refund.               │
│                                                                │
│  ▼ How do I become a seller?                                   │
│    Create a seller account, provide your business license,     │
│    and wait for admin approval (1-3 business days).            │
│                                                                │
│  ▼ What is the platform commission?                            │
│    Shopland takes a 10% commission on each sale.               │
│    Delivery fees are not subject to commission.                │
│                                                                │
│  ▼ Is my personal information safe?                            │
│    Yes. Your national ID and photos are only visible to        │
│    you and Shopland admins. They are never shared with         │
│    sellers or other buyers.                                    │
│                                                                │
│  ── Still need help? ─────────────────────────────────────     │
│  [Contact Support →]                                           │
│                                                                │
└────────────────────────────────────────────────────────────────┘
```

### FAQ Categories

## 4. Terms of Service

### URL: `/terms`

Standard legal page. Sections:
- User eligibility (must be 18+, valid ID)
- Account responsibilities
- Buyer obligations (must pay on delivery)
- Seller obligations (accurate listings, fulfill orders)
- Platform commission terms
- Prohibited items
- Account suspension and termination
- Dispute resolution
- Limitation of liability

---

## 5. Privacy Policy

### URL: `/privacy`

Sections:
- What data we collect (name, ID, phone, address, photos)
- Why we collect it (verification, delivery, communication)
- How we store it (encrypted, secure servers)
- Who can see it (admins only -- never shared with other users publicly)
- User rights (request data deletion, download your data)
- Cookie usage
- Third-party services (SMS gateway, email service)
